# 校验工具

## 1、 是否为 Undefined 类型

```js
this.$m.test.isUndefined('');
```

## 2、 是否为 Null 类型

```js
this.$m.test.isNull('');
```

## 3、 是否为字符串类型

```js
this.$m.test.isString('');
```

## 4、 是否为数字类型

```js
this.$m.test.isNumber('');
```

## 5、 是否为布尔类型

```js
this.$m.test.isBoolean('');
```

## 6、 是否为方法

```js

```

